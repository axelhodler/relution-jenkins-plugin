The path to the icon to display for this application, relative to the workspace directory. If this field is left blank the icon to use is automatically determined from the application's meta-data. If the specified file cannot be found, the field is treated as if it were empty.

Note: This will not change the icon of the application itself. It will only affect the icon that is shown in the app store. Use this if you want the change the icon that is shown in the store to be different from the actual icon of the application.
